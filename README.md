## ğŸ’» **Setup**  

1. **Clone the repository**:  
   ```bash
   git clone https://github.com/DikshitaDas/GitHubber.git
   cd GitHubber
   ```  

2. **Install dependencies**:  
   ```bash
   npm install
   cd client
   npm install
   ```  

3. **Run the app**:  
   ```bash
   # Run the backend  
   npm start  

   # In another terminal, run the frontend  
   cd client  
   npm start  
   ```  

4. **Start exploring GitHub profiles!**
